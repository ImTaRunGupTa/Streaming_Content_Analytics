## 📖 Project Overview

This project presents an **interactive Power BI dashboard** that analyzes content available on **Netflix** and **Amazon Prime Video**.

The dashboard transforms raw streaming platform datasets into meaningful insights through **Power Query**, **DAX**, and interactive visualizations. It enables users to compare content libraries, genres, ratings, release trends, directors, and geographical distribution across two of the world's most popular streaming platforms.

This project demonstrates a complete **Business Intelligence workflow**, including:

- Data Cleaning
- Data Transformation
- Data Modeling
- DAX Calculations
- Interactive Dashboard Design
- Data Visualization
- Cross-platform Content Comparison

# 🎯 Key Insights

- 🎬 Netflix contains a significantly larger content library than Amazon Prime Video.
- 🎭 Drama is the most dominant genre across both streaming platforms.
- 🎥 Movies account for the majority of available content compared to TV Shows.
- 📈 Content production has grown rapidly after 2015.
- 🌍 The United States contributes the highest number of titles on both platforms.
- ⭐ Mature audience ratings (TV-MA / 13+) represent a large share of the available content.

# 🚀 How to Use

### Clone the Repository

```bash
git clone https://github.com/ImTaRunGupTa/Streaming-Content-Analysis.git
```

### Open the Dashboard

1. Install **Microsoft Power BI Desktop**

2. Open:

```text
Dashboard/Streaming Content Dashboard.pbix
```

3. Use the built-in navigation buttons to switch between **Netflix** and **Amazon Prime Video** dashboards.

4. Explore the dashboards using interactive slicers and filters.
