## 🎬 Netflix \& Prime Video Content Analytics Dashboard – Power BI





### 📌 Project Overview



This repository contains a Power BI dashboard designed to analyze and compare content production across Netflix and Amazon Prime Video. The dashboard presents platform-wise insights into the number of movies and web series produced and demonstrates effective use of interactive navigation through action buttons.



The project emphasizes dashboard design, interactivity, and analytical visualization using Power BI.



### 🎯 Project Objectives



* Analyze content production across Netflix and Amazon Prime Video



* Compare the volume of movies and web series by platform



* Enable seamless dashboard navigation using interactive action buttons



* Deliver a clean, professional, and intuitive user experience



### 🛠️ Tools \& Technologies



* Microsoft Power BI Desktop



* Power Query – Data cleaning and transformation



* DAX – Measures and calculated fields
